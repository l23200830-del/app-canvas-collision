# 🔵 Simulador de Colisiones Colectivas en Canvas

Este proyecto es una aplicación web interactiva que simula un entorno físico de **N cuerpos** en movimiento. Utiliza la API de **HTML5 Canvas** para renderizar gráficos 2D y un motor de física basado en vectores para gestionar colisiones elásticas entre círculos de diferentes tamaños.

## 🚀 Características

- **Física Realista:** Implementación de colisiones elásticas siguiendo el modelo de conservación de momentum y energía.
- **Entorno Dinámico:** Generación aleatoria de $N$ círculos con radios, posiciones y velocidades variables.
- **Diseño Moderno:** Interfaz de usuario con estilo *Dark Mode*, fuentes de Google Fonts (Inter) y efectos de *Glassmorphism*.
- **Responsive:** El lienzo se ajusta automáticamente al tamaño de la ventana del navegador.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica y contenedor de dibujo.
- **CSS3:** Estilizado avanzado, degradados y efectos de desenfoque de fondo.
- **JavaScript (ES6+):** Lógica de Programación Orientada a Objetos (POO) para la gestión de partículas y cálculos trigonométricos.
